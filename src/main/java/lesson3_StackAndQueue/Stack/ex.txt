Stack
Первый принцип стека: обратиться в стеке к только к самому верхнему элементу и только так. Добавлять и извлекать только последнюю позицию.
Допустим в стеке 3 элемента. Нужно достать первый элемент, сначала достаем 3, потом 2, потом только 1. Все остальное складываем в стек.
Для чего это нужно? Так мы выстраиваем структуру работы с последними входящими элементами.
У стека есть абревиатура, по которой ее знают везде -> LIFO (last input first output).
